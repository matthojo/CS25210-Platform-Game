Features

Gameplay

  • Randomly generated structures
  • Different difficulty level structures
  • Varied difficulties in spacing beteen structures
  • Lives
  • Coins

Controls

  • Keyboard
  • Mouse
  • Touch (on touch enabled devices)

Scoring

  • Global Highscores, connected to Twitter
  • Local Highscore
  • Ingame highscore placemarkers

Settings

  • Fullscreen Mode (not fully supported in all broswers)
  • Muting
  • Pause
  • Quit Game (reset)
  • Live debugging (left in for the curious)

Extras

  • iOS Device Support (and offline app support)
